# 🌦️ Weather App

A modern and responsive Weather Application developed using HTML, CSS, and JavaScript. The application fetches real-time weather data from the OpenWeather API and provides users with accurate weather information for any city worldwide.

## 🚀 Features

- Search weather by city name
- Real-time temperature updates
- Humidity information
- Wind speed monitoring
- Dynamic weather condition display
- Clean and responsive user interface
- API-based live weather data retrieval

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API

## 📌 Project Overview

This project demonstrates the integration of a third-party REST API with a web application. Users can enter the name of a city and instantly receive current weather details, including temperature, humidity, wind speed, and weather conditions.

The application is designed with a focus on simplicity, responsiveness, and user experience, making it suitable for both desktop and mobile devices.

## ▶️ How to Run the Project

1. Clone the repository:
   ```bash
   git clone https://github.com/sonamsona2005-coder/Weather_App.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Weather_App
   ```

3. Add your OpenWeather API key in the JavaScript file.

4. Open `index.html` in your browser.
